# 🐾 Dr. Paws — AI Veterinary Voice Agent

An AI-powered veterinary voice agent that handles 
animal emergencies and books appointments automatically.

## Features
- Real-time voice conversation
- Animal emergency first aid guidance
- Automatic appointment booking
- Live database integration

## Tech Stack
- LiveKit — Real-time voice
- Groq (Llama 3.3) — AI Brain
- ElevenLabs — Natural voice
- n8n — Automation workflows
- Supabase — Database
- FastAPI — Backend
